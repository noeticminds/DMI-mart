== Description ==
WooCommerce Order Test provides a testing payment gateway for WooCommerce. It allows you to simulate a full checkout process on your site to confirm everything is running smoothly. The gateway is available for admin users only.

== Installation ==
<strong>Install from the WP Dashboard</strong>
  1. Log into your WP dashboard, then click **Plugins** > **Add New**  
  2. Under the "Install Plugins" title, click **Upload**  
  3. Choose the plugin zip file, upload, and activate the plugin.
<strong>Install from FTP</strong>
  1. Extract the zip file and upload the contents to the `wp-content/plugins/` directory of your WP installation.  
  2. Activate the plugin from the Plugins page.
<strong>That's It! You're Done!</strong>
